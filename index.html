🛠 Anleitung: Teams, Spiele und Provider anpassen

1) KickoffAPI als primäre Quelle ändern: suche nach const PROVIDERS = [ ... ]. In V5 steht KickoffAPI bereits an Position 1.

2) Team-Flaggen anpassen: suche nach TEAM_FLAGS.

3) Rankings anpassen: suche nach RANKINGS.

4) Gruppen ändern: suche nach GROUPS.

5) Ergebnisse / kommende Spiele ändern: suche nach SNAPSHOT.completedMatches und SNAPSHOT.upcomingMatches.

6) Bracket ändern: suche nach SNAPSHOT.bracket.

7) Key an einer Stelle setzen: du kannst lokal im Browser speichern oder oberhalb des Hauptscripts eine globale Konfiguration ergänzen.

<script> window.WM_DASH_CONFIG = { kickoffApiKey: "DEIN_KEY_HIER", worldCupApiKey: "OPTIONALER_KEY" }; </script>
Sicherheits-Hinweis: Ich habe deine echte KickoffAPI nicht im auslieferbaren HTML-Code fest verdrahtet, weil die Datei für GitHub Pages öffentlich gehostet werden soll. Ein fest eingebetteter produktiver API-Key in einer öffentlichen HTML-Datei kann missbraucht werden. V5 ist daher so gebaut, dass KickoffAPI trotzdem die primäre Quelle ist und der Key sicher lokal oder über eine separate, nicht veröffentlichte Konfigurationsdatei gesetzt werden kann.
