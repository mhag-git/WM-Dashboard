Modell-Logik

Das Modell kombiniert normiertes FIFA-Ranking mit aktueller Turnierform (Punkte/Spiel, Torverhältnis/Spiel, Tore/Spiel) und berechnet daraus Sieg-/Remis-/Niederlagen-Wahrscheinlichkeiten.

Wichtiger Hinweis

Die Wahrscheinlichkeiten sind eine Heuristik für ein Dashboard – keine offiziellen Wettquoten und kein Ersatz für Echtzeit-Statistik-Feeds.

🛠 Anleitung: Teams und Spiele im Dashboard anpassen

Du kannst die Teams, Spiele und das Bracket direkt in der HTML-Datei bearbeiten. Dafür musst du keine Build-Umgebung starten – ein Texteditor reicht.

1) Teams / FIFA-Rankings anpassen

Suche in der Datei nach const RANKINGS = { ... } und passe Einträge an oder ergänze neue Teams.

const RANKINGS = {
  "Argentina": 1,
  "Germany": 10,
  "Mexico": 14,
  "New Team": 42
};

2) Gruppen anpassen

Suche nach const GROUPS = { ... }. Dort legst du fest, welche Teams in welcher Gruppe auftauchen.

const GROUPS = {
  A: ["Mexico", "South Africa", "Korea Republic", "Czechia"],
  B: ["Canada", "Bosnia and Herzegovina", "Qatar", "Switzerland"]
};

3) Gespielte Ergebnisse ändern

Suche nach SNAPSHOT.completedMatches und bearbeite einzelne Spiele:

{
  id: 1,
  kickoff: "2026-06-11T21:00:00+02:00",
  group: "A",
  home: "Mexico",
  away: "South Africa",
  homeScore: 2,
  awayScore: 0,
  venue: "Mexico City Stadium",
  stage: "group"
}

4) Kommende Spiele ändern

Suche nach SNAPSHOT.upcomingMatches und füge Spiele hinzu oder passe Uhrzeiten an:

{
  id: 41,
  stage: "Gruppenphase",
  group: "J",
  home: "Argentina",
  away: "Austria",
  kickoff: "2026-06-22T19:00:00+02:00",
  venue: "AT&T Stadium, Dallas"
}

5) K.o.-Bracket anpassen

Suche nach SNAPSHOT.bracket. Dort kannst du Platzhalter oder konkrete Teams setzen:

bracket: {
  r32: [
    {
      id: 73,
      code: "M73",
      date: "2026-06-28T21:00:00+02:00",
      labelHome: "A2",
      labelAway: "B2",
      venue: "Los Angeles Stadium"
    }
  ]
}

6) Flaggen anpassen

Suche nach const TEAM_FLAGS = { ... }. Dort kannst du Emoji-Flaggen ergänzen, falls du neue Teams hinzufügst.

7) Auto-Refresh deaktivieren

Wenn du die Live-Abfrage nicht möchtest, kannst du im Browser den Haken bei Auto-Refresh entfernen oder in der HTML-Datei die Checkbox standardmäßig ohne checked setzen.

Wenn der öffentliche Feed nicht erreichbar ist oder CORS blockiert, bleibt das Dashboard trotzdem voll funktionsfähig – dann mit dem integrierten Snapshot als Fallback.
